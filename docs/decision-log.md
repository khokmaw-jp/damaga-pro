# Decision Log

DAMAGA Proサイト運用・開発に関する重要な意思決定を記録する。

## 2026-07-06

- `damaga-pro.jp` は単なる商品LPではなく、企業ホームページ、FAQ/AIO SEO、導入実績、代理店システム、シミュレーションを含む事業基盤として育てる。
- 公開サイトは将来的に Next.js へ移行する。
- 代理店システムは公開サイトと分離し、`partner.damaga-pro.jp` のサブドメインで運用する。
- FAQと導入実績は、ファイル手動更新ではなく管理画面CMSで運用できる形にする。
- シミュレーションは代理店システム内に埋め込み、結果を保存できるようにする。
- ドメイン取得・更新管理はお名前.comで継続する。
- DNS管理、Turnstile、WAFなどはCloudflareで行う方針。
- ソースコードと設計書はGitHubで管理する。
- ホスティングと自動デプロイはVercelを使う方針。

## 保留事項

- DBは Supabase と Neon のどちらを採用するか未決定。
- 認証方式は Supabase Auth または Auth.js/NextAuth を候補とする。
- 画像保存先はCMS実装時に決定する。
- 問い合わせフォームの送信基盤はNext.js移行時に再設計する。
