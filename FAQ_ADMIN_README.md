# DAMAGA PRO FAQ管理

## 初回設定

お名前.comレンタルサーバーの公開ルートにある `.htaccess` へ、次の環境変数を追加します。

```apache
SetEnv FAQ_ADMIN_PASSWORD "十分に長く推測されにくい管理者パスワード"
```

パスワードはHTML、PHP、GitHubへ直接書かないでください。設定後、次のURLからログインします。

https://damaga-pro.jp/faq/admin/

## 管理画面でできること

- Q&Aの追加・編集・削除
- 公開・下書きの切り替え
- カテゴリーの追加・更新
- ハッシュタグの設定
- 関連Q&Aの設定
- title、meta description、本文の更新

初期データとして14カテゴリー、120件のQ&Aを登録しています。各記事には3〜6個のタグと3〜5件の関連Q&Aを設定しています。数値シミュレーション10件は、記事データの `calculator_type` と専用JavaScriptで表示します。

質問への短い回答は、ページ本文とFAQPage JSON-LDの両方へ同じ内容が出力されます。

## サーバー権限

管理画面から追加・更新するには、PHPから次の場所へ書き込める必要があります。

- `faq/data/faqs.json`
- `faq/`（新しいURLフォルダを作成するため）

通常はサーバーへアップロードした所有者権限のままで動作します。保存エラーが出た場合のみ、サーバー管理画面で書き込み権限を確認してください。

## 公開後の確認

1. `/faq/` でカテゴリー絞り込みと検索を確認
2. 各Q&Aページの表示と問い合わせリンクを確認
3. `/faq/admin/` がパスワードなしでは開けないことを確認
4. 新規Q&Aを下書きで作成し、編集後に公開へ切り替える
5. 新しい公開ページが `sitemap.xml` に自動追加されたことを確認する

管理画面から追加・公開したページは公開URLを作成し、`sitemap.xml` にも自動反映します。下書きと削除済みページはサイトマップから除外されます。
