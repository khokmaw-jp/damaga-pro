# DAMAGA Pro Image Library

DAMAGA Pro関連サイトで再利用するための画像素材です。

## Facilities

- `facilities/carehome-exterior.jpg`
  - 推奨用途: 老人ホーム、介護施設、導入モデル
  - 出所: DAMAGA Proサイト制作時の生成画像
  - 注意: 実在施設の導入事例ではなく、施設イメージとして使用

- `facilities/hospital-exterior.jpg`
  - 推奨用途: 病院、医療施設、導入モデル
  - 出所: DAMAGA Proサイト制作時の生成画像
  - 注意: 実在施設の導入事例ではなく、施設イメージとして使用

- `facilities/hospital-exterior-v2.jpg`
  - 推奨用途: 病院、医療施設、導入モデル（推奨版）
  - 特徴: 医療サイン、病院入口、救急車が見え、病院と認識しやすい構図
  - 出所: DAMAGA Proサイト制作時の生成画像
  - 注意: 実在施設の導入事例ではなく、施設イメージとして使用

- `facilities/facility-building.jpg`
  - 推奨用途: ファーストビュー、法人施設、窓の多い建物
  - 出所: ファンビータ既存プロジェクト素材

## Interior

- `interior/window-office.jpg`
  - 推奨用途: 室内環境、日差し、窓際の暑さ、オフィス
  - 出所: ファンビータ既存プロジェクト素材

## Installation

- `installation/film-install.png`
  - 推奨用途: 窓用シート、施工方法、貼るだけ施工
  - 出所: DAMAGA Pro向け提供素材

## Usage Notes

- 生成した施設画像は必ず「施設イメージ」「イメージ写真」などの表記を添えてください。
- 実在施設への導入実績を示す画像としては使用しないでください。
- 既存プロジェクト素材と提供素材を外部広告や第三者案件へ転用する場合は、元の利用条件も確認してください。
- 画像内に文字を焼き込まず、見出しや数値はHTML上で重ねる運用を推奨します。
